usage: check_speed.py [-h] [--serverid SERVERID] [--servername SERVERNAME]
                      [--checkserver CHECKSERVER] [--plot [PLOT]] [--stats]

Check internet speed

options:
  -h, --help            show this help message and exit
  --serverid SERVERID   Preferred server ID
  --servername SERVERNAME
                        Preferred server name (partial match)
  --checkserver CHECKSERVER
                        Check server availability by name and return details
  --plot [PLOT]         Plot internet speed history (last 30 days). Use "text"
                        for terminal plot.
  --stats               Show historical statistics (averages and count)
                        without running a test.
